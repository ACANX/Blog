# Hexo新项目启动命令


### 安装 hexo
```cmd
npm install hexo-cli -g
```

### 初始化项目（blog）

```cmd
hexo init blog
```

### 移动到项目目录

```cmd
cd blog
```

### 登记项目配置

- 略


### 安装依赖

```cmd
npm install
```

### 生成站点内容

```cmd
hexo generate
```

### 启动本地服务器，查验生成的网站内容

```cmd
hexo server
```

### 查看生成的站点

- 访问 [http://localhost:4000/Blog/](http://localhost:4000/Blog/) 查看
