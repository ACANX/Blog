# 万物皆对象

在Java的世界里，万物皆对象包括，普通类、接口类、抽象类、枚举类、Record类（JDK14后支持）

Object 是所有java对象类的公共超父类  ，java.lang.Object    对象是Java对象中最顶层的类，所有的其他对象都是其子类。
